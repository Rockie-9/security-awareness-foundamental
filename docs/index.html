Emerging Risk #1

AI Does Not Forget

Modern AI tools do not understand your intent.
They only understand input.

Common Misjudgment

  • “I didn’t mean to share confidential data.”
  • “It’s just a draft / example.”
  • “I removed the company name.”

Security Reality

  • Context can still be reconstructed
  • Logging / retention may persist
  • Output ≠ safe storage
Rule of thumb:
If data should not leave the company, it should not enter public AI.
(06:20–07:10) Trainer note: focus on “right tool for right data class.”
